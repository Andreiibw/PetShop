Catálogo de Produtos
Descrição: Esta página permite que você explore os produtos disponíveis para compra, filtrando conforme as suas preferências.
Funcionalidades:
•	Navegação no Catálogo: A barra de navegação na parte superior da página permite que você acesse facilmente as categorias principais de produtos: Cachorros, Gatos, Outros Pets, Promoções e Serviços.
•	Filtragem de Produtos: Na coluna à esquerda, você pode aplicar filtros para encontrar produtos específicos. Os filtros incluem:
o	Fragrância Higiene: Opções como Neutro, Cítrico, Talco e Lavanda.
o	Atrativo: Se o produto possui ou não atrativos.
o	Porta de Raça: Tamanho da raça como Pequeno, Médio, Grande, Mini e Gigante.
•	Visualização de Produtos: Os produtos são exibidos com imagem, nome, preço e desconto para assinantes.
Instruções de Uso:
1.	Navegue pelas Categorias: Clique na categoria desejada na barra de navegação.
2.	Aplique Filtros: Marque as caixas de seleção ao lado dos filtros desejados.
3.	Visualize os Produtos: Veja a lista de produtos atualizada conforme os filtros aplicados.

Cadastro de Cliente
Descrição: Esta página permite que novos clientes se registrem no sistema do PetShop.
Funcionalidades:
•	Tipo de Pessoa: Escolha entre Pessoa Física e Pessoa Jurídica.
•	Informações Pessoais: Inclui campos para nome, e-mail, celular, telefone, sexo, data de nascimento e CPF.
•	Senha: Defina uma senha segura que atenda aos requisitos mínimos (mínimo de 8 caracteres, letra minúscula, letra maiúscula e números).
Instruções de Uso:
1.	Preencha o Formulário: Complete todos os campos obrigatórios com suas informações.
2.	Defina uma Senha: Certifique-se de que a senha atende aos requisitos.
3.	Aceite os Termos: Marque a caixa de seleção para concordar com os termos e condições.
4.	Submeta o Formulário: Clique no botão "Cadastrar" para concluir o registro.
________________________________________
Cadastro de Pet
Descrição: Esta página permite que você registre seu pet no sistema do PetShop.
Funcionalidades:
•	Informações do Pet: Campos para nome, idade, peso, raça, sexo, espécie, porte e cor do pet.
Instruções de Uso:
1.	Preencha o Formulário: Complete todos os campos obrigatórios com as informações do seu pet.
2.	Submeta o Formulário: Clique no botão "Cadastrar Pet" para concluir o registro.
________________________________________
Agendamento de Serviços
Descrição: Esta página permite que você agende serviços para seu pet, como banho, tosa, vacinação e consulta.
Funcionalidades:
•	Informações do Cliente: Campos para nome, e-mail e telefone.
•	Data e Hora: Escolha a data e hora do agendamento.
•	Serviço: Selecione o serviço desejado a partir do menu suspenso.
Instruções de Uso:
1.	Preencha o Formulário: Complete todos os campos obrigatórios com suas informações e as informações do agendamento.
2.	Selecione o Serviço: Escolha o serviço desejado.
3.	Submeta o Formulário: Clique no botão "Agendar" para confirmar o agendamento.
